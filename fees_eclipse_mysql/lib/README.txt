Place the MySQL Connector/J JAR in this lib folder and name it exactly:
mysql-connector-java.jar

Steps to get the JAR:
1) Download Connector/J from: https://dev.mysql.com/downloads/connector/j/
2) Extract the ZIP and copy the mysql-connector-java-8.0.xx.jar file into this lib folder.
3) Rename the jar to: mysql-connector-java.jar
4) Refresh the project in Eclipse (right click project -> Refresh)
